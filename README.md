This package provides Jira OAuth 2.0 support for the PHP League's [OAuth 2.0 Client](https://github.com/thephpleague/oauth2-client).

## Installation

To install, use composer:

```
composer require mrjoops/oauth2-jira
```

## Testing

``` bash
$ ./vendor/bin/phpunit
```
